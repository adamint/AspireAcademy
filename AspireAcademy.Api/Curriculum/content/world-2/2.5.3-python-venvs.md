# Virtual Environments: WithPip, WithUv, WithVirtualEnvironment

Configure virtual environment paths and package installation with WithPip and WithUv.

## Why Virtual Environments?

Python projects use virtual environments to isolate dependencies. Without them, all Python apps share the same global packages, leading to version conflicts.

## WithPip — Standard Python Tooling

```csharp
builder.AddPythonApp("api", "../python-api", "app.py")
       .WithPip();
```

This tells Aspire to:
1. Create a virtual environment in `../python-api/.venv` (if it doesn't exist)
2. Run `pip install -r requirements.txt` to install dependencies
3. Activate the virtual environment before running the script

## WithUv — Fast Python Package Manager

```csharp
builder.AddPythonApp("api", "../python-api", "app.py")
       .WithUv();
```

Same behavior as `WithPip`, but uses [uv](https://github.com/astral-sh/uv) — a Rust-based Python package installer that's 10-100x faster than pip. Requires `uv` to be installed on the system.

## Custom Virtual Environment Path

By default, the virtual environment is created in the app directory as `.venv`. Override this:

```csharp
builder.AddPythonApp("api", "../python-api", "app.py")
       .WithVirtualEnvironment("../shared-venv");
```

## Example: FastAPI with Dependencies

Project structure:
```
python-api/
├── main.py
├── requirements.txt
└── .venv/        ← created automatically
```

`requirements.txt`:
```
fastapi>=0.100.0
uvicorn>=0.23.0
```

AppHost:
```csharp
builder.AddUvicornApp("api", "../python-api", "main:app")
       .WithPip()
       .WithHttpEndpoint(targetPort: 8000);
```

Aspire runs:
1. `python -m venv .venv` (create venv)
2. `.venv/bin/pip install -r requirements.txt` (install deps)
3. `.venv/bin/uvicorn main:app` (start app)

## Pip vs Uv

| Feature | WithPip | WithUv |
|---------|---------|--------|
| Speed | Standard | 10-100x faster |
| Requirements | Python only | Requires `uv` CLI |
| Compatibility | Universal | Excellent |

Use `WithPip` for maximum compatibility. Use `WithUv` when you need faster installs (especially in CI).
