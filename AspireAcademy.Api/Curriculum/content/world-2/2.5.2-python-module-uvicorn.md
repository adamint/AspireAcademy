# AddPythonModule and AddUvicornApp

Use AddPythonModule(name, appDirectory, moduleName) for modules and AddUvicornApp(name, appDirectory, app) for ASGI apps.

## AddPythonModule

```csharp
public static IResourceBuilder<PythonAppResource> AddPythonModule(
    this IDistributedApplicationBuilder builder,
    string name,
    string appDirectory,
    string moduleName)
```

Instead of running `python app.py`, this runs `python -m moduleName`. Use it for Python packages that are invoked as modules:

```csharp
builder.AddPythonModule("api", "../python-api", "uvicorn")
       .WithHttpEndpoint(targetPort: 8000);
```

This executes: `python -m uvicorn`

## AddUvicornApp

```csharp
public static IResourceBuilder<UvicornAppResource> AddUvicornApp(
    this IDistributedApplicationBuilder builder,
    string name,
    string appDirectory,
    string app)
```

Purpose-built for ASGI frameworks (FastAPI, Starlette). The `app` parameter is the ASGI application path:

```csharp
builder.AddUvicornApp("api", "../fastapi-app", "main:app")
       .WithHttpEndpoint(targetPort: 8000, env: "UVICORN_PORT");
```

This executes: `uvicorn main:app`

## AddPythonExecutable

```csharp
public static IResourceBuilder<PythonAppResource> AddPythonExecutable(
    this IDistributedApplicationBuilder builder,
    string name,
    string appDirectory,
    string executableName)
```

Runs a Python-installed executable (from `pip install`):

```csharp
builder.AddPythonExecutable("celery", "../worker", "celery")
       .WithArgs("worker", "--app=tasks");
```

## Choosing the Right Method

| Method | Runs | Best For |
|--------|------|----------|
| `AddPythonApp` | `python script.py` | Flask, Django manage.py, custom scripts |
| `AddPythonModule` | `python -m module` | Any pip-installed module |
| `AddUvicornApp` | `uvicorn app:module` | FastAPI, Starlette, ASGI frameworks |
| `AddPythonExecutable` | Installed executable | Celery, Gunicorn, other CLI tools |

## FastAPI Example

```python
# main.py
from fastapi import FastAPI
app = FastAPI()

@app.get("/")
def read_root():
    return {"message": "Hello from FastAPI!"}
```

```csharp
builder.AddUvicornApp("api", "../fastapi-app", "main:app")
       .WithPip()
       .WithHttpEndpoint(targetPort: 8000);
```
