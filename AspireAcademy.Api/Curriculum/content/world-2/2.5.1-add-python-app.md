# AddPythonApp — Python Script Resources

Use AddPythonApp(name, appDirectory, scriptPath) to run Python scripts as Aspire resources.

## The API

```csharp
public static IResourceBuilder<PythonAppResource> AddPythonApp(
    this IDistributedApplicationBuilder builder,
    string name,
    string appDirectory,
    string scriptPath)
```

- **`name`** — resource name for Dashboard and service discovery
- **`appDirectory`** — path to the Python project directory (resolved relative to AppHost)
- **`scriptPath`** — the Python script to run (e.g., `app.py`)

## Basic Example

```csharp
var builder = DistributedApplication.CreateBuilder(args);

var api = builder.AddPythonApp("flask-api", "../python-api", "app.py")
                 .WithHttpEndpoint(targetPort: 5000, env: "PORT");

builder.AddProject<Projects.Web>("web")
       .WithReference(api);

builder.Build().Run();
```

## How It Works

The orchestrator:
1. Finds the Python executable on the system PATH
2. Sets the working directory to `appDirectory`
3. Runs `python scriptPath` with injected environment variables
4. Manages the process lifecycle (start, health check, stop)

## Injecting Configuration

Python apps receive Aspire configuration through environment variables:

```csharp
builder.AddPythonApp("api", "../python-api", "app.py")
       .WithHttpEndpoint(targetPort: 8000, env: "PORT")
       .WithEnvironment("DB_CONNECTION", db.Resource.ConnectionStringExpression);
```

Your Python code:

```python
import os
port = int(os.environ.get("PORT", 8000))
db_conn = os.environ.get("DB_CONNECTION")
```

## Virtual Environments

By default, Aspire uses the system Python. To use a virtual environment, chain `WithPip()` or `WithUv()`:

```csharp
builder.AddPythonApp("api", "../python-api", "app.py")
       .WithPip()  // Install from requirements.txt using pip
       .WithHttpEndpoint(targetPort: 8000);
```

This creates/activates a virtual environment and installs dependencies before starting the app.

## Flask Example

```python
# app.py
from flask import Flask
import os

app = Flask(__name__)

@app.route("/")
def hello():
    return {"message": "Hello from Python!"}

if __name__ == "__main__":
    port = int(os.environ.get("PORT", 5000))
    app.run(host="0.0.0.0", port=port)
```
