# Building Aspire from Source

Everything you need to clone, build, and run the Aspire test suite locally.

---

## Prerequisites

Check `global.json` for the exact .NET SDK version:

```bash
cat global.json | grep version
# e.g., "version": "10.0.100-preview.3"
```

You also need:
- **Docker** (or Podman) for integration tests
- **Node.js** for the VS Code extension and JavaScript tests
- **Git** for cloning

## Clone and Build

```bash
git clone https://github.com/dotnet/aspire.git
cd aspire

# Restore and build everything
./build.sh       # Linux/macOS
.\build.cmd      # Windows

# Or build specific projects
dotnet build src/Aspire.Hosting/Aspire.Hosting.csproj
dotnet build src/Aspire.Hosting.Redis/Aspire.Hosting.Redis.csproj
dotnet build src/Aspire.Cli/Aspire.Cli.csproj
```

## Running Tests

```bash
# All tests (slow — starts Docker containers)
dotnet test

# Specific integration
dotnet test tests/Aspire.Hosting.Redis.Tests/

# Skip Docker-dependent tests
dotnet test --filter "Category!=RequiresDocker"

# App model tests only (fast, no Docker)
dotnet test tests/Aspire.Hosting.Tests/ \
    --filter "FullyQualifiedName~AppModel"
```

## Building the CLI

```bash
dotnet build src/Aspire.Cli/Aspire.Cli.csproj
# Install locally for testing from built artifacts
./build.sh --pack && cp artifacts/packages/*.tar.gz ~/.aspire/bin/
```

## Building the Dashboard

```bash
dotnet build src/Aspire.Dashboard/Aspire.Dashboard.csproj
dotnet run --project src/Aspire.Dashboard/
```

## Building the VS Code Extension

```bash
cd extension
npm install
npm run build
# Launch in VS Code: F5 (opens Extension Development Host)
```

## Running Playground Samples

```bash
cd playground/TypeScriptAppHost
aspire run

cd playground/TypeScriptAppHost
aspire run
```

## Common Build Issues

| Issue | Fix |
|---|---|
| Wrong SDK version | Install the version from `global.json` |
| Docker not running | Start Docker Desktop / daemon |
| Port conflicts | DCP allocates random ports, but some tests expect specific ones |
| NuGet restore fails | Run `dotnet restore` with the repo's NuGet.config |
| Node.js not found | Install Node.js for extension and JS tests |
