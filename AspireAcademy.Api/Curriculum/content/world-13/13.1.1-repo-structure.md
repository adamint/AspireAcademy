# Solution Layout and Project Map

The `dotnet/aspire` repository contains the entire Aspire platform — hosting SDK, integrations, Dashboard, CLI, VS Code extension, polyglot type system, and code generators. Understanding its structure is the first step to navigating the codebase.

---

## Top-Level Layout

```
aspire/
  src/                    Source projects (shipping code)
  tests/                  Test projects (100+)
  playground/             Sample apps for manual testing
  extension/              VS Code extension (TypeScript)
  eng/                    Build engineering (CI, props, targets)
  docs/                   Contributor documentation
  global.json             SDK version pinning
  Aspire.slnx             Solution file
```

## Core Hosting (src/Aspire.Hosting/)

The heart of Aspire. This is the SDK that every AppHost uses:

```
src/Aspire.Hosting/
  ApplicationModel/         IResource, annotations, model, notification service
  Dcp/                      DCP process management, K8s client, resource types
  Eventing/                 IDistributedApplicationEventing, event types
  Pipelines/                PipelineStep, pipeline executor
  Publishing/               Container runtime, manifest publisher
  Health/                   Health check evaluation loop
  DistributedApplication.cs          CreateBuilder() entry point
  DistributedApplicationBuilder.cs   Builder implementation
```

## Hosting Integrations (src/Aspire.Hosting.*)

53 integration packages. Each follows the same structure:

```
src/Aspire.Hosting.Redis/
  RedisResource.cs
  RedisBuilderExtensions.cs

src/Aspire.Hosting.PostgreSQL/
  PostgresServerResource.cs
  PostgresDatabaseResource.cs
  PostgresBuilderExtensions.cs
```

Key packages: Redis, PostgreSQL, SqlServer, MongoDB, RabbitMQ, Kafka, Nats, Azure (28 sub-packages), Kubernetes, Docker, RemoteHost, JavaScript, Python, Orleans, Yarp, Keycloak, Seq, OpenAI, GitHub.Models, Foundry, DevTunnels.

## Dashboard (src/Aspire.Dashboard/)

Blazor Server application:

```
src/Aspire.Dashboard/
  Components/           Blazor pages and components
  ServiceClient/        gRPC client (IDashboardClient)
  Otlp/                 OTLP receiver and telemetry storage
  Model/                View models (ResourceViewModel, etc.)
  Configuration/        Dashboard settings
```

## CLI (src/Aspire.Cli/)

```
src/Aspire.Cli/
  Commands/             40+ command implementations
  Mcp/                  MCP server and tools
  Agents/               Agent auto-detection (Claude, Copilot, etc.)
  Utils/                Environment checker, project locator
  Backchannel/          JSON-RPC client to running AppHosts
```
