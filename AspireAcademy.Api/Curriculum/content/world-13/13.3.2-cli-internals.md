# CLI Internals

The Aspire CLI is a .NET tool built with `System.CommandLine`. It has 40+ commands organized hierarchically.

---

## BaseCommand

Every command inherits from `BaseCommand`:

```csharp
// src/Aspire.Cli/Commands/BaseCommand.cs

internal abstract class BaseCommand : Command
{
    protected IInteractionService InteractionService { get; }
    protected AspireCliTelemetry Telemetry { get; }
    protected CliExecutionContext ExecutionContext { get; }

    protected BaseCommand(string name, string description,
        IFeatures features, ICliUpdateNotifier updateNotifier,
        CliExecutionContext executionContext,
        IInteractionService interactionService,
        AspireCliTelemetry telemetry)
        : base(name, description)
    {
        SetAction(async (parseResult, cancellationToken) =>
        {
            _executionContext.Command = this;

            // Execute the command
            var exitCode = await ExecuteAsync(parseResult, ct);

            // Check for CLI updates (non-blocking)
            if (UpdateNotificationsEnabled)
                updateNotifier.NotifyIfUpdateAvailable();

            return exitCode;
        });
    }

    protected abstract Task<int> ExecuteAsync(
        ParseResult parseResult, CancellationToken ct);
}
```

## RunCommand Flow

```csharp
// src/Aspire.Cli/Commands/RunCommand.cs (key flow)

protected override async Task<int> ExecuteAsync(
    ParseResult parseResult, CancellationToken ct)
{
    // 1. Locate the AppHost project
    var project = await _projectLocator.FindProjectAsync(ct);

    // 2. Set up completion sources
    var buildComplete = new TaskCompletionSource<bool>();
    var backchannelReady = new TaskCompletionSource<IAppHostCliBackchannel>();

    // 3. Start the AppHost process
    var pendingRun = project.RunAsync(new AppHostProjectContext
    {
        BuildCompletionSource = buildComplete,
        BackchannelCompletionSource = backchannelReady,
    }, ct);

    // 4. Wait for build
    var buildSuccess = await buildComplete.Task.WaitAsync(ct);

    // 5. Wait for backchannel connection
    var backchannel = await backchannelReady.Task.WaitAsync(ct);

    // 6. Get Dashboard URLs
    var dashboardUrls = await backchannel.GetDashboardUrlsAsync(ct);

    // 7. Display resource status until Ctrl+C
    await DisplayResourceStatusAsync(backchannel, ct);
}
```

## Agent Environment Scanners

Each scanner detects a specific AI agent:

```
src/Aspire.Cli/Agents/
  ClaudeCode/ClaudeCodeAgentEnvironmentScanner.cs
  VsCode/VsCodeAgentEnvironmentScanner.cs
  OpenCode/OpenCodeAgentEnvironmentScanner.cs
  Playwright/PlaywrightSupport.cs
  CopilotCli/CopilotCliRunner.cs
```

Scanners check for:
- Environment variables set by the agent process
- Configuration files in expected locations
- Parent process names
