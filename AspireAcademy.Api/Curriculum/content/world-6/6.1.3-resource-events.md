# Resource-Scoped Events

While global events cover the entire application lifecycle, resource-scoped events let you react to individual resources starting, becoming healthy, or stopping. This precision is critical for tasks like database seeding or service-to-service orchestration.

## BeforeResourceStartedEvent

`BeforeResourceStartedEvent` fires just before a specific resource begins its startup process. This is the canonical place to seed databases, run migrations, or inject configuration that must exist before the resource serves traffic.

```csharp
var db = builder.AddPostgres("pg").AddDatabase("mydb");

builder.Eventing.Subscribe<BeforeResourceStartedEvent>(
    db.Resource,
    async (evt, ct) =>
    {
        var logger = evt.Services.GetRequiredService<ILogger<Program>>();
        logger.LogInformation("Seeding database '{Name}'...", evt.Resource.Name);

        // Run EF migrations or SQL scripts here
    });
```

The key detail is the second parameter to `Subscribe` — passing `db.Resource` ensures this handler only fires for the `mydb` database, not for every resource in the application.

## ResourceReadyEvent

`ResourceReadyEvent` signals that a resource has passed all configured health checks and is genuinely ready to accept connections. This is safer than `BeforeResourceStartedEvent` for dependent services that need a live connection.

```csharp
var cache = builder.AddRedis("cache");

builder.Eventing.Subscribe<ResourceReadyEvent>(
    cache.Resource,
    async (evt, ct) =>
    {
        // Cache is healthy — pre-warm it
        var logger = evt.Services.GetRequiredService<ILogger<Program>>();
        logger.LogInformation("Redis is ready, pre-warming cache...");
    });
```

## Combining Scoped Events

You can subscribe to multiple events on the same resource to build ordered workflows: use `BeforeResourceStartedEvent` for setup, then `ResourceReadyEvent` to verify everything is working. Each handler receives the `IServiceProvider` via `evt.Services`, giving you full access to registered services for any work you need to perform.
