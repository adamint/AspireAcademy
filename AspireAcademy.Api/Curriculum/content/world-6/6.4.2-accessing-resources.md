# Accessing Resources in Tests

Once your `DistributedApplication` is running, you need to interact with its resources — querying endpoints, reading connection strings, and waiting for services to become ready before executing assertions.

## Getting Endpoints

Use `GetEndpoint` to retrieve the allocated URL for a named resource:

```csharp
var endpoint = app.GetEndpoint("apiservice");
// Returns something like "http://localhost:54321"
```

Since the testing builder randomizes ports, you should never hardcode URLs. Always resolve endpoints dynamically through the application instance.

## Retrieving Connection Strings

For infrastructure resources like databases and caches, use `GetConnectionStringAsync` to obtain the resolved connection string:

```csharp
var connectionString = await app.GetConnectionStringAsync("postgres");
```

This returns the fully resolved string with actual hostnames and ports, which you can use to create direct database connections for verifying state in your tests.

## Waiting for Readiness

Resources take time to start — containers must pull images, databases must initialize, and services must bind to ports. The `WaitForHealthyAsync` extension method blocks until a resource reports healthy status:

```csharp
await app.ResourceNotifications
    .WaitForResourceHealthyAsync("apiservice");
```

This is critical for test reliability. Without readiness checks, your HTTP calls or database queries may hit services that are still initializing, causing intermittent test failures.

## Asserting Database State

Combine connection string retrieval with direct database access to verify side effects:

```csharp
var connStr = await app.GetConnectionStringAsync("catalogdb");
await using var conn = new NpgsqlConnection(connStr);
await conn.OpenAsync();
var count = await conn.ExecuteScalarAsync<int>(
    "SELECT COUNT(*) FROM products");
Assert.True(count > 0);
```

This pattern lets you verify that API operations actually persisted data correctly, going beyond HTTP response validation to confirm real infrastructure state.
