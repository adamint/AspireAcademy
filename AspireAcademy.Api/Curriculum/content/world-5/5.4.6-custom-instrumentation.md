# Custom Instrumentation for Shared Libraries

When building reusable libraries or shared components, you need instrumentation that is self-contained and discoverable by consuming applications. This lesson covers patterns for creating production-quality custom instrumentation.

## Defining a Reusable ActivitySource

Place the `ActivitySource` in your library with a versioned name following reverse-domain conventions.

```csharp
// In MyCompany.Payments library
internal static class PaymentTelemetry
{
    internal static readonly ActivitySource Source =
        new("MyCompany.Payments", "1.0.0");

    internal static readonly Meter Meter =
        new("MyCompany.Payments", "1.0.0");
}
```

## Wrapping Operations

Create spans around meaningful operations, using `ActivityKind` to classify the work.

```csharp
public async Task<PaymentResult> ProcessPaymentAsync(PaymentRequest request)
{
    using var activity = PaymentTelemetry.Source.StartActivity(
        "ProcessPayment", ActivityKind.Client);

    activity?.SetTag("payment.provider", "stripe");
    activity?.SetTag("payment.amount", request.Amount);
    activity?.SetTag("payment.currency", request.Currency);

    try
    {
        var result = await CallPaymentProvider(request);
        activity?.SetTag("payment.transaction_id", result.TransactionId);
        activity?.SetStatus(ActivityStatusCode.Ok);
        return result;
    }
    catch (Exception ex)
    {
        activity?.SetStatus(ActivityStatusCode.Error, ex.Message);
        throw;
    }
}
```

## Consumer Registration

Applications consuming the library register its sources in their OpenTelemetry configuration.

```csharp
builder.Services.AddOpenTelemetry()
    .WithTracing(t => t.AddSource("MyCompany.Payments"))
    .WithMetrics(m => m.AddMeter("MyCompany.Payments"));
```

## Testing with ActivityListener

Verify instrumentation in unit tests without an OTLP exporter.

```csharp
var activities = new List<Activity>();
using var listener = new ActivityListener
{
    ShouldListenTo = s => s.Name == "MyCompany.Payments",
    Sample = (ref ActivityCreationOptions<ActivityContext> _) =>
        ActivitySamplingResult.AllDataAndRecorded,
    ActivityStopped = a => activities.Add(a)
};
ActivitySource.AddActivityListener(listener);

await service.ProcessPaymentAsync(request);

Assert.Single(activities);
Assert.Equal("ProcessPayment", activities[0].OperationName);
```
