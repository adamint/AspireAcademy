# Keycloak, Seq, and YARP

Aspire has hosting integrations for identity management, structured logging, and reverse proxying.

---

## Keycloak — Identity Management

```csharp
var keycloak = builder.AddKeycloak("identity")
    .WithDataVolume("keycloak-data");

builder.AddProject<Projects.Api>("api")
    .WithReference(keycloak);
```

- **Image**: `quay.io/keycloak/keycloak:latest`
- **Ports**: `8080` (HTTP), `8443` (HTTPS)
- Provides OpenID Connect, SAML, and OAuth2
- Admin console for realm and user management

## Seq — Structured Logging

```csharp
var seq = builder.AddSeq("seq")
    .WithLifetime(ContainerLifetime.Persistent);
```

- **Image**: `datalust/seq:latest`
- **Port**: `5341` (ingestion), `80` (web UI)
- Queryable structured log storage
- Integrates with OpenTelemetry (logs flow from Dashboard → Seq)

## YARP — Reverse Proxy

```csharp
var api = builder.AddProject<Projects.Api>("api");
var frontend = builder.AddViteApp("frontend", "../web");

builder.AddYarp("gateway")
    .WithReference(api)
    .WithReference(frontend)
    .LoadFromConfiguration("ReverseProxy");
```

YARP (Yet Another Reverse Proxy) is Microsoft's high-performance .NET reverse proxy. Use it as an API gateway in front of multiple services.
