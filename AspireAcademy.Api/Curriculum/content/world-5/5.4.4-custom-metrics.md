# Custom Metrics

Metrics provide quantitative measurements of your application's behavior over time. Aspire integrations register built-in metrics automatically, but custom metrics let you track business-specific indicators like orders placed, revenue processed, or queue depths.

## Creating a Meter

Define a `Meter` per logical component, similar to `ActivitySource` for tracing.

```csharp
public static class OrderMetrics
{
    public static readonly Meter Meter = new("MyApp.Orders");

    public static readonly Counter<long> OrdersPlaced =
        Meter.CreateCounter<long>("orders.placed", "orders",
            "Total number of orders placed");

    public static readonly Histogram<double> OrderDuration =
        Meter.CreateHistogram<double>("orders.duration_ms", "ms",
            "Time to process an order");
}
```

## Recording Measurements

Counters increment monotonically. Histograms record distributions of values. Both support tags for dimensional breakdowns.

```csharp
app.MapPost("/orders", async (Order order) =>
{
    var sw = Stopwatch.StartNew();

    await ProcessOrder(order);

    sw.Stop();
    OrderMetrics.OrdersPlaced.Add(1,
        new KeyValuePair<string, object?>("region", order.Region),
        new KeyValuePair<string, object?>("tier", order.CustomerTier));

    OrderMetrics.OrderDuration.Record(sw.Elapsed.TotalMilliseconds,
        new KeyValuePair<string, object?>("region", order.Region));

    return Results.Created($"/orders/{order.Id}", order);
});
```

## Registering with OpenTelemetry

Like `ActivitySource`, custom meters must be registered for export.

```csharp
builder.Services.AddOpenTelemetry()
    .WithMetrics(metrics =>
    {
        metrics.AddMeter("MyApp.Orders");
    });
```

## Dashboard Visualization

In the dashboard, navigate to **Metrics → Resource → Meter → Instrument**. Counter charts show cumulative totals and per-second rates. Histograms display percentile breakdowns (p50, p90, p99). Tags appear as filterable dimensions, letting you slice metrics by region, customer tier, or any other tag you define.
