# RabbitMQ, Kafka, and NATS Clients

Register messaging clients for RabbitMQ, Kafka, and NATS with health checks and telemetry.

Aspire provides client integrations for the most popular message brokers. Each follows the same `Add` pattern, registering the native client with health checks, tracing, and metrics.

## RabbitMQ

The `Aspire.RabbitMQ.Client` package registers `IConnection`:

```csharp
builder.AddRabbitMQClient("messaging");
```

Use it to create channels and publish messages:

```csharp
public class OrderPublisher(IConnection connection)
{
    public async Task PublishAsync(OrderCreated order)
    {
        using var channel = await connection.CreateChannelAsync();
        await channel.BasicPublishAsync(
            exchange: "orders",
            routingKey: "order.created",
            body: JsonSerializer.SerializeToUtf8Bytes(order));
    }
}
```

Configuration key: `Aspire:RabbitMQ:Client`.

## Kafka

The `Aspire.Confluent.Kafka` package registers `IProducer<TKey, TValue>` and `IConsumer<TKey, TValue>`:

```csharp
builder.AddKafkaProducer<string, string>("kafka");
builder.AddKafkaConsumer<string, string>("kafka");
```

```csharp
public class EventProducer(IProducer<string, string> producer)
{
    public async Task SendAsync(string topic, string key, string value)
        => await producer.ProduceAsync(topic, new Message<string, string>
        {
            Key = key, Value = value
        });
}
```

## NATS

The `Aspire.NATS.Net` package registers `INatsConnection`:

```csharp
builder.AddNatsClient("nats");
```

```csharp
public class NotificationService(INatsConnection nats)
{
    public async Task NotifyAsync(string subject, string data)
        => await nats.PublishAsync(subject, data);
}
```

## AppHost Wiring

All three follow the same hosting pattern:

```csharp
var rabbit = builder.AddRabbitMQ("messaging");
var kafka = builder.AddKafka("kafka");
var nats = builder.AddNats("nats");

builder.AddProject<Projects.Worker>()
    .WithReference(rabbit)
    .WithReference(kafka)
    .WithReference(nats);
```

Each integration auto-registers a health check that verifies broker connectivity at startup.
