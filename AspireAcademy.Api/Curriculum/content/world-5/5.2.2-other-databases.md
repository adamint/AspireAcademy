# MySQL, MongoDB, Oracle, Cosmos DB

Beyond PostgreSQL and SQL Server, Aspire has hosting integrations for every major database.

---

## MySQL

```csharp
var mysql = builder.AddMySql("mysql")
    .WithPhpMyAdmin();  // adds phpMyAdmin container
var db = mysql.AddDatabase("shop");
```

- **Image**: `mysql:9`
- **Port**: `3306`
- **Connection string**: `Server={host};Port={port};Database=shop;Uid=root;Pwd={generated}`

## MongoDB

```csharp
var mongo = builder.AddMongoDB("mongo")
    .WithMongoExpress(); // adds Mongo Express UI
var db = mongo.AddDatabase("analytics");
```

- **Image**: `mongo:8`
- **Port**: `27017`
- **Connection string**: `mongodb://{host}:{port}/analytics`

MongoDB is popular with Python and Node.js services. The connection string works with `pymongo` (Python), `mongodb` driver (Node.js), and `mongo-driver` (Go).

> 🌍 **Real World:** MongoDB is especially popular in startups and rapid prototyping because its schema-less design lets you iterate quickly without migrations. Many analytics pipelines also use MongoDB for storing semi-structured event data.

## Oracle

```csharp
var oracle = builder.AddOracle("oracle");
var db = oracle.AddDatabase("hr");
```

- **Image**: `container-registry.oracle.com/database/free:latest`
- **Port**: `1521`

## Azure Cosmos DB (Emulator)

```csharp
var cosmos = builder.AddAzureCosmosDB("cosmos")
    .RunAsEmulator();
```

Runs the Cosmos DB emulator locally for development. In publish mode, provisions an Azure Cosmos DB account.

## The Universal Pattern

Regardless of database:

```csharp
builder.AddProject<Projects.Api>("api")
    .WithReference(db)
    .WaitFor(db);
```

The API receives `ConnectionStrings__<name>` and uses `WaitFor` to ensure the database is ready. This works identically whether `db` is PostgreSQL, MongoDB, or Cosmos DB.
