# MySQL Client Integration

The `Aspire.MySqlConnector` package uses the [MySqlConnector](https://mysqlconnector.net/) library — a high-performance, async-first ADO.NET provider — to connect your services to MySQL databases.

## Registration

```csharp
builder.AddMySqlDataSource("mysqldb");
```

This registers `MySqlDataSource` as a singleton, similar to the Npgsql pattern. The data source manages connection pooling internally and is fully thread-safe.

For multiple MySQL instances, use keyed registration:

```csharp
builder.AddKeyedMySqlDataSource("orders");
builder.AddKeyedMySqlDataSource("inventory");
```

## Usage

```csharp
public class OrderService(MySqlDataSource dataSource)
{
    public async Task<int> GetOrderCountAsync()
    {
        await using var conn = await dataSource.OpenConnectionAsync();
        await using var cmd = conn.CreateCommand();
        cmd.CommandText = "SELECT COUNT(*) FROM orders";
        return Convert.ToInt32(await cmd.ExecuteScalarAsync());
    }
}
```

## Settings and Configuration

The settings class is `MySqlConnectorSettings`, configured under `Aspire:MySqlConnector`:

```json
{
  "Aspire": {
    "MySqlConnector": {
      "DisableHealthChecks": false,
      "DisableTracing": false,
      "DisableMetrics": false
    }
  }
}
```

```csharp
builder.AddMySqlDataSource("mysqldb", settings =>
{
    settings.DisableHealthChecks = true;
});
```

## Observability

The integration wires up all three observability pillars automatically:

- **Health check**: Registered as `"MySql"` — verifies database connectivity
- **Tracing**: Captured via `AddSource("MySqlConnector")` — every SQL command generates a trace span
- **Metrics**: Exposed via `AddMeter("MySqlConnector")` — connection pool and command execution statistics

## MySqlConnector vs. MySQL.Data

Aspire uses MySqlConnector (not Oracle's MySQL.Data) because it offers true async I/O, better performance, and built-in OpenTelemetry support. The `MySqlDataSource` pattern mirrors `NpgsqlDataSource`, giving you a consistent experience across PostgreSQL and MySQL integrations.
