# Entity Framework Core Integration

Register EF Core DbContext with AddDbContext patterns and automatic retry policies.

For applications using Entity Framework Core, Aspire provides specialized integrations that register your `DbContext` with connection pooling, automatic retries, health checks, and telemetry baked in.

## Registering a DbContext

Use the database-specific EF Core method in your service's `Program.cs`:

```csharp
// PostgreSQL via Npgsql EF Core provider
builder.AddNpgsqlDbContext<CatalogContext>("catalogdb");

// SQL Server
builder.AddSqlServerDbContext<OrderContext>("orderdb");
```

These methods register your `DbContext` in DI and configure the underlying provider with Aspire-optimized defaults, including retry policies for transient failures.

## Defining Your DbContext

Define your context as usual — no Aspire-specific changes needed:

```csharp
public class CatalogContext : DbContext
{
    public CatalogContext(DbContextOptions<CatalogContext> options)
        : base(options) { }

    public DbSet<Product> Products => Set<Product>();
}
```

Inject and use it in your services:

```csharp
public class CatalogService(CatalogContext db)
{
    public async Task<List<Product>> GetAllAsync()
        => await db.Products.ToListAsync();
}
```

## Configuration

Configure via `Aspire:Microsoft:EntityFrameworkCore:Npgsql` (or `:SqlServer`):

```json
{
  "Aspire": {
    "Microsoft": {
      "EntityFrameworkCore": {
        "Npgsql": {
          "DisableHealthChecks": false,
          "DisableTracing": false,
          "DisableRetry": false
        }
      }
    }
  }
}
```

Or use inline delegates:

```csharp
builder.AddNpgsqlDbContext<CatalogContext>("catalogdb",
    settings => settings.DisableRetry = false);
```

## Automatic Retry

By default, the EF Core integrations configure **execution strategies** that automatically retry on transient database errors. This is critical in cloud environments where brief connectivity blips are expected. Set `DisableRetry = true` only if you manage retries yourself.
