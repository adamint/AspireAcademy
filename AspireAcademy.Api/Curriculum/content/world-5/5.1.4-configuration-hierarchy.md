# Configuration Hierarchy

Aspire client integrations support a five-layer configuration precedence model. Each layer can override the previous one, giving you fine-grained control from global defaults down to per-instance overrides.

## The Five Layers (Last Wins)

1. **Compiled defaults** — Built-in values in the settings class (e.g., `DisableHealthChecks = false`)
2. **Global component section** — `Aspire:{Component}` applies to all instances of that integration
3. **Named instance section** — `Aspire:{Component}:{connectionName}` targets a specific connection
4. **ConnectionStrings entry** — `ConnectionStrings:{name}` provides the connection string
5. **Inline delegate** — `configureSettings` callback has the highest precedence

## JSON Configuration Example

```json
{
  "ConnectionStrings": {
    "catalogdb": "Host=localhost;Database=catalog",
    "analyticsdb": "Host=localhost;Database=analytics"
  },
  "Aspire": {
    "Npgsql": {
      "DisableHealthChecks": false,
      "DisableTracing": false,
      "catalogdb": {
        "DisableTracing": true
      }
    }
  }
}
```

In this example, `catalogdb` has tracing disabled (layer 3 overrides layer 2), while `analyticsdb` inherits the global setting with tracing enabled.

## Inline Delegate Override

The `configureSettings` delegate always wins, regardless of what JSON configuration says:

```csharp
// This overrides ALL JSON/environment configuration for health checks
builder.AddNpgsqlDataSource("catalogdb", settings =>
{
    settings.DisableHealthChecks = true;
});
```

## Why This Matters

This hierarchy enables practical deployment patterns:

- **Development**: Enable all diagnostics globally via `appsettings.Development.json`
- **Production**: Disable verbose tracing globally, but enable it for critical paths via named sections
- **Testing**: Override everything with inline delegates for deterministic test behavior

Understanding precedence prevents surprises. If a health check isn't registering, check whether a higher-precedence layer disabled it. The delegate (layer 5) is the definitive source of truth — if it sets a value, nothing else can override it.
