# Agent Integration

Aspire auto-detects AI agent environments and configures the MCP server for each one.

---

## Supported Agents

| Agent | Scanner |
|---|---|
| **Claude Code** | `ClaudeCodeAgentEnvironmentScanner` |
| **VS Code Copilot** | `VsCodeAgentEnvironmentScanner` |
| **OpenCode** | `OpenCodeAgentEnvironmentScanner` |
| **Playwright** | `PlaywrightSupport` |
| **GitHub Copilot CLI** | `CopilotCliRunner` |

## How Auto-Detection Works

When you run `aspire agent mcp`, the CLI:

1. Scans environment variables and parent process information
2. Each `AgentEnvironmentScanner` checks for its agent's indicators
3. The matching scanner configures the MCP session (tool filtering, output formatting)
4. Default configuration is used if no agent is detected

## Using Aspire with Claude Code

```
User: "What resources are running?"
Claude: [calls list_resources]
        "6 resources: pg (Running), cache (Running), api (Running),
         frontend (Running), ml (FailedToStart), worker (Running)"

User: "Why did ml fail?"
Claude: [calls list_console_logs("ml")]
        "ModuleNotFoundError: No module named 'scikit-learn'"

User: "Show me the slowest trace"
Claude: [calls list_traces]
        "Trace abc123: GET /search took 3.2s
         Bottleneck: ML service query at 2.8s"
```

## Using Aspire with VS Code Copilot

The extension auto-registers the MCP server. In Copilot Chat:

```
@workspace What Aspire resources are running?
@workspace Show me the logs from the API
@workspace Search the Aspire docs for health checks
```

Copilot uses list_resources, list_console_logs, and search_docs MCP tools.

## Agent Scanner Architecture

```
src/Aspire.Cli/Agents/
    CommonAgentApplicators.cs    - shared logic
    ClaudeCode/                  - Claude detection
    VsCode/                      - VS Code detection
    OpenCode/                    - OpenCode detection
    Playwright/                  - Playwright detection
    CopilotCli/                  - GitHub Copilot CLI
```

CommonAgentApplicators handles tool registration, backchannel setup, and notification routing. Agent-specific scanners handle detection and formatting.
