# Extension Features

The Aspire VS Code extension transforms your editor into an Aspire-aware development environment.

---

## Code Lenses

**Run | Debug** buttons appear above the AppHost entry point in every supported language:

```csharp
// Run | Debug
var builder = DistributedApplication.CreateBuilder(args);
```

```typescript
// Run | Debug
const builder = await createBuilder();
```

```python
# Run | Debug
with create_builder() as builder:
```

Clicking Run starts `aspire run` in the integrated terminal. Clicking Debug attaches the VS Code debugger.

## Tree View

The Aspire sidebar panel shows:

1. **Detected AppHosts** — all AppHost files found in your workspace
2. **Resources** — every resource, grouped by type (containers, projects, executables)
3. **Dependencies** — which resources reference which
4. **Running State** — green/red indicators when an AppHost is running

The tree updates automatically when you save your AppHost file.

## Debugger Integration

The extension provides a custom debug adapter:

- Set breakpoints in AppHost code
- Step through resource registration (AddRedis, AddProject, etc.)
- Inspect builder and resource objects
- View annotations added by extension methods
- Debug configurations are auto-generated (no manual launch.json)

## Gutter Decorations

When an AppHost is running, resource lines get colored indicators:

| Decoration | Meaning |
|---|---|
| Green dot | Running and healthy |
| Red dot | Failed to start |
| Gray dot | Not started or stopped |
| Blue dot | Starting |

## MCP Server Registration

The extension automatically registers the Aspire MCP server with VS Code Copilot Chat (`AspireMcpServerDefinitionProvider`). Copilot gains access to Aspire MCP tools when you are working in an Aspire workspace — no manual configuration required.

## Walkthrough

Built-in walkthrough accessible from the Welcome tab:
1. Install the Aspire CLI (stable or daily build)
2. Verify with `aspire doctor` via the extension
3. Create a project with `aspire new`
4. Run and explore the Dashboard
