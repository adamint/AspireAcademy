# Extension Features

The Aspire VS Code extension provides IDE integration for developing Aspire applications.

---

## Code Lenses

Run/Debug buttons appear above AppHost entry points:

```csharp
// ▶ Run | 🐛 Debug
var builder = DistributedApplication.CreateBuilder(args);
```

Click "Run" to start the AppHost directly from the editor.

## Tree View

The Aspire sidebar shows:
- Detected AppHosts in your workspace
- Resources defined in each AppHost
- Running instance status

## Debugger Integration

The extension provides a custom debug adapter:
- Set breakpoints in your AppHost code
- Step through resource registration
- Inspect the app model during startup

## Gutter Decorations

Resource names in your AppHost code get colored indicators:
- 🟢 Resource is running
- 🔴 Resource failed to start
- ⚪ Resource not started

## Walkthrough

The extension includes a walkthrough guide:
1. Install the CLI (stable or daily build)
2. Verify with `aspire doctor`
3. Create your first project
4. Run and explore the Dashboard
