# AI-Powered Secure Application

Build an AppHost combining identity, vector search, AI, and logging services.

## Your Mission

Architect an AI application with secure authentication, vector search for context retrieval, OpenAI for chat, and Seq for comprehensive logging.

## Requirements

1. Add Keycloak `"idp"` with realm import from `"./realms"` and data volume
2. Add Qdrant `"vectors"` with data volume
3. Add OpenAI `"openai"` with a `"chat"` model using `"gpt-4o"`
4. Add Seq `"seq"` with data volume
5. Add `"ai-api"` project referencing all four resources (Keycloak, Qdrant, OpenAI chat model, Seq)

## Hints

- Keycloak realm: `AddKeycloak("idp").WithRealmImport("./realms").WithDataVolume()`
- OpenAI model: `openai.AddModel("chat", "gpt-4o")` — reference the model, not the parent
- Qdrant auto-generates an API key if you don't supply one
