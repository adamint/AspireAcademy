# Orleans: AddOrleans

Use AddOrleans(name) to configure an Orleans service with clustering, grain storage, and streaming.

## What is Orleans?

Orleans is a .NET framework for building distributed, scalable applications using the virtual actor (grain) model. Aspire provides hosting support to configure Orleans clustering and storage.

## Basic Setup

```csharp
var storage = builder.AddRedis("storage");
var orleans = builder.AddOrleans("orleans")
                     .WithClustering(storage)
                     .WithGrainStorage("Default", storage);

builder.AddProject<Projects.Silo>("silo")
       .WithReference(orleans)
       .WithReference(storage);
```

## Key Configuration

- **Clustering** — how silo instances discover each other
- **Grain Storage** — where grain state is persisted
- **Streaming** — event streaming between grains

## Client Access

```csharp
builder.AddProject<Projects.Api>("api")
       .WithReference(orleans.AsClient());
```

The `.AsClient()` method configures the project as an Orleans client rather than a silo.
