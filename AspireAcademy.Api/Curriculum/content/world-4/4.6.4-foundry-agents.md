# Foundry Hosted Agents

Use AsHostedAgent, RunAsHostedAgent, and PublishAsHostedAgent to deploy AI agents to Foundry.

## Hosted Agent Pattern

Aspire supports deploying AI agents as hosted services in Azure AI Foundry:

```csharp
var foundry = builder.AddFoundry("ai");

builder.AddProject<Projects.AgentService>("agent")
       .AsHostedAgent(foundry);
```

## RunAsHostedAgent

For local development, run the agent locally while connecting to Foundry:

```csharp
builder.AddProject<Projects.AgentService>("agent")
       .RunAsHostedAgent(foundry);
```

## PublishAsHostedAgent

For deployment, publish the agent to Foundry's hosting infrastructure:

```csharp
builder.AddProject<Projects.AgentService>("agent")
       .PublishAsHostedAgent(foundry);
```

## Agent Architecture

Hosted agents are containerized services that:
- Receive requests through Foundry's routing
- Access AI models via the Foundry deployment
- Scale automatically based on demand
- Integrate with Foundry's monitoring and evaluation tools

This is an advanced pattern for production AI agent deployments.
