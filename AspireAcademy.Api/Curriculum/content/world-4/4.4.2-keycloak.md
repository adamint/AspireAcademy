# Keycloak: AddKeycloak

Add Keycloak for identity management with AddKeycloak(name) and configure realms.

## Basic Setup

```csharp
var keycloak = builder.AddKeycloak("keycloak");

builder.AddProject<Projects.Api>("api")
       .WithReference(keycloak);
```

Keycloak provides OAuth 2.0 / OpenID Connect authentication. The container starts with an admin console for configuring realms, clients, and users.

## Import Realm Configuration

```csharp
builder.AddKeycloak("keycloak")
       .WithRealmImport("./realms");
```

This imports pre-configured realm JSON files on startup, useful for reproducible development environments.

## With Data Volume

```csharp
builder.AddKeycloak("keycloak")
       .WithDataVolume()
       .WithLifetime(ContainerLifetime.Persistent);
```

Persists Keycloak configuration and user data between restarts.
