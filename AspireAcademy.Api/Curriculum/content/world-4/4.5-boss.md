# Polyglot Microservices Platform

Build an AppHost for a polyglot microservices platform combining .NET, Python, JavaScript, and multiple infrastructure resources.

## Your Mission

Design a full-stack platform with:
- PostgreSQL for data persistence
- Redis for caching with persistence
- A .NET API service
- A Python ML service
- A React frontend
- A YARP gateway for unified routing

## Requirements

1. Add PostgreSQL `"pg"` with database `"appdb"` and data volume
2. Add Redis `"cache"` with persistence
3. Add .NET project `"dotnet-api"` referencing PostgreSQL and Redis
4. Add Python app `"ml-service"` at `"../python-ml"` running `"app.py"`, referencing PostgreSQL
5. Add JavaScript app `"frontend"` at `"../react-app"` with `"dev"` script, referencing the .NET API
6. Add YARP `"gateway"` with routes to the .NET API and ML service

## Hints

- Python app: `AddPythonApp("ml-service", "../python-ml", "app.py")`
- JavaScript app: `AddJavaScriptApp("frontend", "../react-app", "dev")`
- Use `WithReference(dotnetApi)` on the frontend to inject the API URL
- YARP routes use `WithConfiguration` to set up path-based routing
