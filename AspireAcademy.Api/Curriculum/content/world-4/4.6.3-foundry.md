# Foundry: AddFoundry and AddDeployment

Use AddFoundry(name) to create a Foundry resource, then AddDeployment(name, modelName, modelVersion, format) for model deployments.

## Basic Setup

```csharp
var foundry = builder.AddFoundry("ai");
var deployment = foundry.AddDeployment("chat", "gpt-4o", "2024-08-06");

builder.AddProject<Projects.Api>("api")
       .WithReference(deployment);
```

Azure AI Foundry (formerly Azure AI Services) provides managed AI model endpoints. `AddFoundry` creates the Foundry resource, and `AddDeployment` configures a specific model deployment.

## Client-Side

```csharp
builder.AddAzureOpenAIClient("chat");
```

## Multiple Deployments

```csharp
var foundry = builder.AddFoundry("ai");
var chat = foundry.AddDeployment("chat", "gpt-4o", "2024-08-06");
var embedding = foundry.AddDeployment("embed", "text-embedding-ada-002", "2");

builder.AddProject<Projects.Api>("api")
       .WithReference(chat)
       .WithReference(embedding);
```
