# Foundry: AddFoundry and AddDeployment

Use `AddFoundry(name)` to create a Foundry resource, then `AddDeployment(name, modelName, modelVersion, format)` for model deployments, or use `FoundryModel` constants for a simpler API.

## Basic Setup

```csharp
var foundry = builder.AddFoundry("ai");

// Using FoundryModel constants (recommended):
var deployment = foundry.AddDeployment("chat", FoundryModel.OpenAI.Gpt5Mini);

// Or using explicit parameters (format is required):
var deployment = foundry.AddDeployment("chat", "gpt-4o", "2024-08-06", "OpenAI");

builder.AddProject<Projects.Api>("api")
       .WithReference(deployment);
```

The `format` parameter specifies the model format (e.g., `"OpenAI"`). The package is `Aspire.Hosting.Foundry`.

## Client-Side

```csharp
builder.AddAzureOpenAIClient("chat");
```

## Multiple Deployments

```csharp
var foundry = builder.AddFoundry("ai");
var chat = foundry.AddDeployment("chat", FoundryModel.OpenAI.Gpt5Mini);
var embedding = foundry.AddDeployment("embed", "text-embedding-ada-002", "2", "OpenAI");

builder.AddProject<Projects.Api>("api")
       .WithReference(chat)
       .WithReference(embedding);
```

## Foundry Projects

For more advanced scenarios, create a Foundry project for AI agents:

```csharp
var foundry = builder.AddFoundry("ai");
var project = foundry.AddProject("myproject");
var model = project.AddModelDeployment("chat", FoundryModel.OpenAI.Gpt5Mini);
```
