# RabbitMQ: AddRabbitMQ

Add a RabbitMQ server with AddRabbitMQ(name) and configure management UI access.

## Basic Setup

```csharp
var rabbit = builder.AddRabbitMQ("messaging");

builder.AddProject<Projects.Api>("api")
       .WithReference(rabbit);

builder.AddProject<Projects.Worker>("worker")
       .WithReference(rabbit)
       .WaitFor(rabbit);
```

Injects: `ConnectionStrings__messaging = amqp://guest:guest@localhost:5672`

## Management Plugin

Add the RabbitMQ Management UI (web dashboard on port 15672):

```csharp
builder.AddRabbitMQ("messaging")
       .WithManagementPlugin();
```

## Custom Credentials

```csharp
var username = builder.AddParameter("rabbit-user");
var password = builder.AddParameter("rabbit-password", secret: true);

builder.AddRabbitMQ("messaging", userName: username, password: password);
```

## Client-Side Usage

```csharp
builder.AddRabbitMQClient("messaging");

// Publishing
public class OrderPublisher(IConnection connection)
{
    public async Task PublishOrder(Order order)
    {
        using var channel = await connection.CreateChannelAsync();
        await channel.BasicPublishAsync("orders", "", body);
    }
}
```

## Health Checks

`AddRabbitMQ` registers health checks that verify the broker accepts connections. Services using `WaitFor` will wait until RabbitMQ is healthy.
