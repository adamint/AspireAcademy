# Redis: AddRedis

Add a Redis cache with AddRedis(name) and configure persistence with WithDataVolume.

## Basic Setup

```csharp
var redis = builder.AddRedis("cache");

builder.AddProject<Projects.Api>("api")
       .WithReference(redis)
       .WaitFor(redis);
```

Injects: `ConnectionStrings__cache = localhost:6379`

## With Persistence

```csharp
builder.AddRedis("cache")
       .WithDataVolume()
       .WithLifetime(ContainerLifetime.Persistent);
```

## Redis Commander

Add a web UI for browsing Redis data:

```csharp
builder.AddRedis("cache")
       .WithRedisCommander();
```

## Redis Insight

Alternative web UI:

```csharp
builder.AddRedis("cache")
       .WithRedisInsight();
```

## Custom Password

```csharp
var password = builder.AddParameter("redis-password", secret: true);
builder.AddRedis("cache", password: password);
```

## Client-Side Usage

In your service project:

```csharp
builder.AddRedisClient("cache");

// Then inject IConnectionMultiplexer
public class CacheService(IConnectionMultiplexer redis)
{
    public async Task SetAsync(string key, string value)
    {
        var db = redis.GetDatabase();
        await db.StringSetAsync(key, value);
    }
}
```

## Health Checks

`AddRedis` registers health checks automatically. Redis reports healthy when it responds to PING commands.
