# NATS: AddNats

Add NATS with AddNats(name) and configure JetStream for persistent messaging.

## Basic Setup

```csharp
var nats = builder.AddNats("nats");

builder.AddProject<Projects.Api>("api")
       .WithReference(nats)
       .WaitFor(nats);
```

Injects: `ConnectionStrings__nats = nats://localhost:4222`

## With JetStream

Enable persistent messaging with JetStream:

```csharp
builder.AddNats("nats")
       .WithJetStream();
```

## Client-Side

```csharp
builder.AddNatsClient("nats");
```

## When to Use NATS

NATS excels at lightweight, low-latency messaging. It's simpler than Kafka and lighter than RabbitMQ. Use it for real-time communication, microservice events, and pub/sub patterns where simplicity matters.

| Feature | NATS | RabbitMQ | Kafka |
|---------|------|----------|-------|
| Complexity | Low | Medium | High |
| Persistence | JetStream | Built-in | Built-in |
| Latency | Very low | Low | Low |
| Ordering | Per subject | Per queue | Per partition |
