# Kafka: AddKafka

Add Apache Kafka with AddKafka(name) and configure topics and broker settings.

## Basic Setup

```csharp
var kafka = builder.AddKafka("kafka");

builder.AddProject<Projects.Api>("api")
       .WithReference(kafka)
       .WaitFor(kafka);
```

Injects: `ConnectionStrings__kafka = localhost:9092`

## With Kafka UI

```csharp
builder.AddKafka("kafka")
       .WithKafkaUI();
```

## Client-Side Usage

```csharp
// Producer
builder.AddKafkaProducer<string, string>("kafka");

// Consumer
builder.AddKafkaConsumer<string, string>("kafka", config =>
{
    config.GroupId = "my-consumer-group";
});
```

## Key Characteristics

Kafka is designed for high-throughput, ordered event streaming. Unlike RabbitMQ, Kafka retains messages for a configurable period, allowing consumers to replay events.
