# Oracle: AddOracleDatabase

Add an Oracle Database container with AddOracleDatabase(name) and AddDatabase(databaseName).

## Basic Setup

```csharp
var oracle = builder.AddOracleDatabase("oracle");
var db = oracle.AddDatabase("mydb");

builder.AddProject<Projects.Api>("api")
       .WithReference(db)
       .WaitFor(db);
```

## Custom Password

```csharp
var password = builder.AddParameter("oracle-password", secret: true);
var oracle = builder.AddOracleDatabase("oracle", password: password);
```

## Container Image

`AddOracleDatabase` uses the Oracle Database Free container image. Oracle containers are larger than PostgreSQL or SQL Server, so the first pull may take longer.

## Persistent Storage

```csharp
builder.AddOracleDatabase("oracle")
       .WithDataVolume()
       .WithLifetime(ContainerLifetime.Persistent);
```

## Connection String

The injected connection string follows Oracle's format:

```
ConnectionStrings__mydb = user id=system;password=...;data source=localhost:1521/FREEPDB1
```

## When to Use

Use `AddOracleDatabase` when your application requires Oracle-specific features (PL/SQL, Oracle-specific SQL syntax) or when you're developing against an existing Oracle production database.
