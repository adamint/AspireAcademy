# Milvus: AddMilvus

Add a Milvus vector database with AddMilvus(name) for high-performance vector operations.

## Basic Setup

```csharp
var milvus = builder.AddMilvus("milvus");

builder.AddProject<Projects.Api>("api")
       .WithReference(milvus)
       .WaitFor(milvus);
```

Milvus is a cloud-native vector database designed for billion-scale similarity search. It supports multiple index types and distance metrics.

## With Persistent Storage

```csharp
builder.AddMilvus("milvus")
       .WithDataVolume()
       .WithLifetime(ContainerLifetime.Persistent);
```

## Milvus vs Qdrant

| Feature | Milvus | Qdrant |
|---------|--------|--------|
| Scale | Billion-scale | Million-scale |
| Architecture | Distributed | Single-node or distributed |
| Index types | IVF, HNSW, DiskANN | HNSW |
| Best for | Large-scale enterprise | Simpler deployments |
