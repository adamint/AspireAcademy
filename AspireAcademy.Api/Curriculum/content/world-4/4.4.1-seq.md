# Seq: AddSeq

Add Seq for structured log aggregation with AddSeq(name) and its web UI.

## Basic Setup

```csharp
var seq = builder.AddSeq("seq");

builder.AddProject<Projects.Api>("api")
       .WithReference(seq);
```

Seq provides a web UI for searching and analyzing structured logs. It complements the Aspire Dashboard by offering advanced log querying, alerting, and retention.

## Accessing the UI

Seq's web interface is available on the endpoint shown in the Dashboard. Use it to write structured queries across all your services' log output.
