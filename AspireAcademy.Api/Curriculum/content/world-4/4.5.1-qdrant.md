# Qdrant: AddQdrant

Add a Qdrant vector database with AddQdrant(name) for similarity search workloads.

## Basic Setup

```csharp
var qdrant = builder.AddQdrant("qdrant");

builder.AddProject<Projects.Api>("api")
       .WithReference(qdrant)
       .WaitFor(qdrant);
```

Qdrant is a vector similarity search engine. Use it for semantic search, recommendation systems, and RAG (Retrieval-Augmented Generation) with AI models.

## With Persistent Storage

```csharp
builder.AddQdrant("qdrant")
       .WithDataVolume()
       .WithLifetime(ContainerLifetime.Persistent);
```

## Client-Side

```csharp
builder.AddQdrantClient("qdrant");
```

The Qdrant client lets you create collections, insert vectors, and perform similarity searches.
