# MySQL and MongoDB

Add MySQL with AddMySql(name) and MongoDB with AddMongoDB(name) including replica sets.

## MySQL

```csharp
var mysql = builder.AddMySql("mysql");
var db = mysql.AddDatabase("mydb");

builder.AddProject<Projects.Api>("api")
       .WithReference(db)
       .WaitFor(db);
```

Injects: `ConnectionStrings__mydb = Server=localhost;Port=3306;Database=mydb;User Id=root;Password=...`

### phpMyAdmin

```csharp
builder.AddMySql("mysql")
       .WithPhpMyAdmin();
```

## MongoDB

```csharp
var mongo = builder.AddMongoDB("mongo");
var db = mongo.AddDatabase("mydb");

builder.AddProject<Projects.Api>("api")
       .WithReference(db)
       .WaitFor(db);
```

Injects: `ConnectionStrings__mydb = mongodb://localhost:27017/mydb`

### Mongo Express

```csharp
builder.AddMongoDB("mongo")
       .WithMongoExpress();
```

### Persistent Storage

Both support data volumes:

```csharp
builder.AddMySql("mysql")
       .WithDataVolume()
       .WithLifetime(ContainerLifetime.Persistent);

builder.AddMongoDB("mongo")
       .WithDataVolume()
       .WithLifetime(ContainerLifetime.Persistent);
```

## Comparison

| Feature | MySQL | MongoDB |
|---------|-------|---------|
| Model | Relational (SQL) | Document (NoSQL) |
| Connection string | Standard ADO.NET format | MongoDB URI format |
| Web UI | phpMyAdmin | Mongo Express |
| .NET client | MySqlConnector | MongoDB.Driver |
