# OpenAI: AddOpenAI

Add an OpenAI resource for language model access with connection string injection.

## Basic Setup

```csharp
var openai = builder.AddConnectionString("openai");

builder.AddProject<Projects.Api>("api")
       .WithReference(openai);
```

Configure the connection string in user secrets or appsettings:

```json
{
  "ConnectionStrings": {
    "openai": "Endpoint=https://api.openai.com;Key=sk-..."
  }
}
```

## With Azure OpenAI

For Azure-hosted models, use the Azure OpenAI integration:

```csharp
var openai = builder.AddAzureOpenAI("openai");
openai.AddDeployment("gpt-4o", "gpt-4o", "2024-08-06");
```

## Client-Side

```csharp
builder.AddOpenAIClientFromConfiguration("openai");

// Or for Azure
builder.AddAzureOpenAIClient("openai");
```

## Usage in Code

```csharp
public class ChatService(OpenAIClient client)
{
    public async Task<string> Chat(string prompt)
    {
        var chatClient = client.GetChatClient("gpt-4o");
        var response = await chatClient.CompleteChatAsync(prompt);
        return response.Value.Content[0].Text;
    }
}
```
