# GitHub Models: AddGitHubModel

Use `AddGitHubModel(name, model)` to access models from the GitHub Models catalog. The method accepts either a model name string or a `GitHubModel` static field.

## Basic Setup

```csharp
var model = builder.AddGitHubModel("ai", "openai/gpt-4o");

builder.AddProject<Projects.Api>("api")
       .WithReference(model);
```

GitHub Models provides access to AI models through your GitHub account. The connection details are injected automatically.

## Using GitHubModel Constants

```csharp
var model = builder.AddGitHubModel("ai", GitHubModel.OpenAIGpt4o);
```

`GitHubModel` provides static fields for well-known models. The field names follow the pattern `{Provider}{Model}` — e.g., `GitHubModel.OpenAIGpt4o`.

## Optional Organization Parameter

Both overloads accept an optional `organization` parameter for GitHub organization-scoped access:

```csharp
var org = builder.AddParameter("github-org");
var model = builder.AddGitHubModel("ai", GitHubModel.OpenAIGpt4o, organization: org);
```

## Client-Side

Services receive the model configuration via connection string and can use the OpenAI-compatible SDK:

```csharp
builder.AddOpenAIClientFromConfiguration("ai");
```

## Available Models

GitHub Models catalog includes models from OpenAI, Meta, Microsoft, and others. Check the GitHub Models marketplace for the current list.
