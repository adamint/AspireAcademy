# GitHub Models: AddGitHubModel

Use AddGitHubModel(name, model) or AddGitHubModel(name, GitHubModel) to access models from the GitHub Models catalog.

## Basic Setup

```csharp
var model = builder.AddGitHubModel("ai", "gpt-4o");

builder.AddProject<Projects.Api>("api")
       .WithReference(model);
```

GitHub Models provides access to AI models through your GitHub account. The connection details are injected automatically.

## Using the Enum

```csharp
var model = builder.AddGitHubModel("ai", GitHubModel.Gpt4o);
```

## Client-Side

Services receive the model configuration via connection string and can use the OpenAI-compatible SDK:

```csharp
builder.AddOpenAIClientFromConfiguration("ai");
```

## Available Models

GitHub Models catalog includes models from OpenAI, Meta, Microsoft, and others. Check the GitHub Models marketplace for the current list.
