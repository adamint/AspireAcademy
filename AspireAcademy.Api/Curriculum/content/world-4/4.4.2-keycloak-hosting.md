# Keycloak & Realm Management

Keycloak is an open-source identity and access management server. Aspire's integration lets you run a local Keycloak instance for development authentication and authorization.

## Adding Keycloak

The `AddKeycloak` extension registers a `KeycloakResource` backed by `quay.io/keycloak/keycloak:26.5`:

```csharp
var keycloak = builder.AddKeycloak("idp");
```

**Signature:**
```csharp
IResourceBuilder<KeycloakResource> AddKeycloak(
    this IDistributedApplicationBuilder builder,
    string name,
    int? port = null,
    IResourceBuilder<ParameterResource>? adminUsername = null,
    IResourceBuilder<ParameterResource>? adminPassword = null)
```

Default port is **8080**. Admin credentials are auto-generated if not supplied.

## Endpoint URL

Keycloak implements `IResourceWithServiceDiscovery`, so `WithReference` injects endpoint URLs (not connection strings):
```
services__idp__http__0=http://{host}:{port}
```

Services use this URL for OIDC configuration, token validation, and user management. With Aspire's service discovery, services can reference Keycloak using `http://idp` in their configuration.

## Realm Import

Pre-configure Keycloak with realm JSON files using `WithRealmImport`:

```csharp
var keycloak = builder.AddKeycloak("idp")
    .WithRealmImport("./keycloak-realms");
```

Place realm export JSON files in the specified directory. Keycloak imports them on startup, setting up:
- Clients (applications)
- Users and roles
- Identity providers
- Authentication flows

## Feature Toggles

Enable or disable Keycloak features:

```csharp
var keycloak = builder.AddKeycloak("idp")
    .WithEnabledFeatures("admin-fine-grained-authz")
    .WithDisabledFeatures("impersonation");
```

## Telemetry Integration

Export Keycloak's own telemetry to the Aspire Dashboard:

```csharp
var keycloak = builder.AddKeycloak("idp")
    .WithOtlpExporter();
```

## Data Persistence

```csharp
var keycloak = builder.AddKeycloak("idp")
    .WithDataVolume();
```

Persists realm configuration, users, and sessions across restarts.

## Complete Example

```csharp
var builder = DistributedApplication.CreateBuilder(args);

var keycloak = builder.AddKeycloak("idp")
    .WithRealmImport("./realms")
    .WithDataVolume();

builder.AddProject<Projects.WebApi>("web-api")
    .WithReference(keycloak);

builder.Build().Run();
```

## Key Takeaways

- **Resource type**: `KeycloakResource` (container, implements `IResourceWithServiceDiscovery`).
- **Container image**: `quay.io/keycloak/keycloak:26.5`.
- **Endpoint URL**: Uses `services__` injection pattern (not `ConnectionStrings__`) for OIDC integration.
- **`WithRealmImport`**: Pre-configure realms, clients, and users from JSON files.
- **`WithOtlpExporter`**: Sends Keycloak's telemetry to the Aspire Dashboard.
- **Dev auth**: Eliminates the need for cloud identity providers during development.
