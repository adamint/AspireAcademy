# aspire new — Pick Your Language

The `aspire new` command scaffolds a new Aspire project. You choose the template and Aspire generates the project structure.

---

## Available Templates

| Template | Description |
|---|---|
| `aspire-starter` | Full .NET starter with AppHost, ServiceDefaults, API, and web frontend |
| `aspire-empty` | Minimal AppHost — just the entry point |
| `aspire-apphost` | AppHost project only (for adding to existing solutions) |
| `aspire-apphost-singlefile` | Single-file AppHost using `apphost.cs` |
| `aspire-ts-cs-starter` | TypeScript + C# combo starter |
| `aspire-py-starter` | Python starter |
| `aspire-js-frontend-starter` | JavaScript frontend starter |

## Scaffolding in Different Languages

### C# (Single-File)

```bash
aspire new aspire-apphost-singlefile --name Atlas
```

This creates an `apphost.cs` file with directives:

```csharp
#:sdk Aspire.AppHost
#:package Aspire.Hosting.Redis

var builder = DistributedApplication.CreateBuilder(args);
var cache = builder.AddRedis("cache");
builder.Build().Run();
```

### TypeScript

```bash
aspire new aspire-starter --name Atlas --language typescript
```

This creates an `apphost.ts` file:

```typescript
import { createBuilder } from './.modules/aspire.js';

const builder = await createBuilder();
const cache = await builder.addRedis("cache");
await builder.build().run();
```

### Python

```bash
aspire new aspire-py-starter --name Atlas
```

This creates an `apphost.py` file:

```python
from aspire_app import create_builder

with create_builder() as builder:
    cache = builder.add_redis("cache")
    builder.run()
```

## What Gets Generated

Regardless of language, `aspire new` creates:

1. **The AppHost file** — your app model
2. **Generated SDK modules** — language-specific type-safe bindings (`.modules/aspire.js` for TypeScript, `aspire_app/` for Python)
3. **Configuration files** — `aspire.config.json`, `appsettings.json`, etc.

The generated SDK modules give you autocomplete, type checking, and documentation right in your editor.
