# Boss: Build a Two-Service App

You've learned how to create an Aspire project, add resources with `AddProject<T>()`, wire dependencies with `WithReference()`, and launch everything with `Build().Run()`. Now it's time to put it all together.

In this boss challenge, you'll build a complete two-service Aspire application from scratch — an API backend and a Web frontend, connected so the frontend can discover and call the API.

This is the fundamental Aspire pattern you'll use in every project going forward. Complete the challenge below to prove your mastery of Module 1.2.
